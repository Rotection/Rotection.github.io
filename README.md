# Rotection

Community-driven Roblox game safety rating system  
Focused on honesty, fairness, and protecting players.

## What is Rotection?

Rotection is a project designed to help players check if a Roblox game
is safe, respectful, and age-appropriate â€” rated by the community.

It does **not** judge gameplay quality.  
It only evaluates safety & fairness.

## How Ratings Work

1. Players submit a rating through the Rotection Discord forum
2. They must follow a strict template
3. Rotection parses the post and verifies structure
4. Scores get calculated and added to the website
5. Mods only step in to remove spam or jokes  
   *no manual approval for every rating*

## Why This Exists

Roblox has millions of games.  
Not all are safe for everyone.

Rotection gives parents & kids a **simple trust system** without judgment,  
just facts and scores from users.
